# 💫 About Me:
I'm a passionate and easy-going backend developer from Nigeria with experience in C# and ASP.NET Core technologies. I love building strong APIs, writing clean code, and creating meaningful applications. In addition to the usual software development, I'm also on a mission to develop my knowledge in the intersection of technology and healthcare. I'm presently on a mission to develop my knowledge in bioinformatics, data science, and machine learning, with an emphasis on the application of these fields in the healthcare sector. Some technologies I'm familiar with in this space include R programming for statistical computing and data visualization, as well as Python-based machine learning libraries. I'm motivated by the potential for technology to revolutionize the way healthcare is delivered. I'm passionate about building technologies that seek to close the gap between software engineering and life sciences. This includes the development of decision support systems, the analysis of genomic data, as well as the building of predictive models for patient outcomes.<br>
